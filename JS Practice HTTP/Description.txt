В ходе подготовки к выполнению задания я столкнулся с проблемой получения ключа от API, т.к. для этого необходима банковская карта.
В результате было принято решение найти ключ на каком-нибудь из форумов. Он был найден, однако лишь для 2.5 версии API.

Далее, в ходе выполнения задания были проверены различные эндпоинты и изучена документация. В результате, удалось получить лишь ответы
из группы 4xx и 2xx (итоги на скриншотах). Для оставшихся же групп ответы получить не удалось. 

Однако, было принято решение написать собственную обертку над API, которая и позволила вернуть коды из 1xx, 3xx, 5xx групп (server.js).
